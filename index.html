JS Generative Art

This is a generative art project using the P5 JavaScript library made by Matt and Beto.

The code is structured using conventional programing principles by seperating JS files by concern. We used three files, one for the background, one for the foreground and one as a joiner.

The P5 Draw() function is called 60 times per second as default for animation and that is where we instantiate the background and foreground class objects. We decided to use class objects to allow for a cleaner sepertion of concerns.

Check out the repository for details: https://github.com/betodute/generative-art-collaborative-project
