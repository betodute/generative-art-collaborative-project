Playful Shapes is a generative art project made by Matthew and Beto using the P5 JavaScript library.

Play around by pressing the buttons to create a combination of shapes and colors that you like. The foreground shapes respond to the position of your cursor.

For more details and code, please check out our GitHub repository. Special thanks to The Coding Train, Omnibus-Type and Gene Kogan for inspiration, education, and code snippets.
